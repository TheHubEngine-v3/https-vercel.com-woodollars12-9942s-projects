## FREESTACK VIDEO — WHAT WAS BUILT

FreeStack Video is John's own self-hosted video render server.
It completely replaces Shotstack with zero cost and zero watermarks.

**Files built:**
- `src/server.js`   — Express REST API (POST /render, GET /render/:id)
- `src/renderer.js` — FFmpeg engine (scenes → MP4 with text overlays)
- `src/storage.js`  — Cloudflare R2 upload (returns public video URL)
- `package.json`    — Node.js dependencies
- `Dockerfile`      — Ubuntu + FFmpeg + fonts pre-installed
- `render.yaml`     — Render.com auto-deploy config

**How it works:**
1. Hub Engine sends JSON timeline (scenes with title, text, CTA, duration)
2. FreeStack builds FFmpeg command with drawtext overlays
3. FFmpeg renders MP4 at 720p, ultrafast preset
4. Video uploads to Cloudflare R2
5. Public URL returned — no watermark, no cost, no limits

**API format (same as before, no API key needed):**
POST https://freestack-video.onrender.com/render
{ timeline: { scenes: [...] } }
→ returns { id, status, url }
