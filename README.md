## What Is The Hub Engine?

The Hub Engine v3 is a fully mobile-optimized, single-file AI business platform built for content creators, affiliate marketers, and job seekers. It runs entirely in the browser — no app store, no installation, no monthly fees beyond the free Groq API.

Built by a 22-year chemical plant and production operations professional who taught himself AI from scratch during a career transition. Every feature solves a real problem.

## ✨ Key Features

### 🤖 AI Video Agents — Susan · Alisha · Lisa · Paul
Four AI avatar agents that each write their own weekly video scripts and generate 8 ready-to-use scene prompts automatically:
- **Susan** — Monday AI News Anchor
- **Alisha** — Wednesday AI News Reporter  
- **Lisa** — Friday AI News + Affiliate Host
- **Paul** — Thursday Podcast Host

One tap → script written → prompts copied → paste into free video tool → done.

**Free video tools supported:** Grok Aurora · Kling AI (66 free credits/day) · Hailuo AI · PixVerse  
**Free clip joiners:** Vmake AI · InVideo · Dreamina · LTX Studio

---

### 🎙️ Becky Chat — Creative Director
- Full AI voice conversation using built-in browser TTS
- Live Mic mode — completely hands-free voice chat
- File and image upload with vision AI
- Persistent conversation memory across sessions

### 💬 Claude Chat — AI Developer
- Same voice and live mic features as Becky
- Focused on app building, business strategy, and technical help

### 💼 Jobs Tab — SE Texas Career Search
- 7 pre-filtered job search links for Chemical Plant · Operations · Supervisor roles
- Targeted for Southeast Texas (Beaumont · Orange · Port Arthur area)
- AI Career Coach with resume upload for personalized cover letters

### 🔗 Affiliate Tab
- Quick links to Vidnoz (50% commission), InVideo (30%), ClickBank, Digistore24, Amazon, ShareASale, CJ Affiliate, JVZoo
- AI marketing expert chat for writing promo scripts and ad copy

### 📱 Social Media Hub
- One-tap access to TikTok, YouTube, Instagram, Facebook, LinkedIn, X, Snapchat, Discord, Telegram, Substack
- AI social media assistant for captions and content strategy

### 🧰 AI Tools Hub
- Quick links to Claude, Gemini, ChatGPT, Grok, Perplexity, DeepSeek, Meta AI, Kling AI, HeyGen, NotebookLM, Replit, Sora

### 📚 Learn Tab
- Free AI courses: Andrew Ng, Google AI Essentials, Anthropic, Elements of AI
- Curated YouTube 30-minute tutorials on Claude, AI Agents, Prompt Engineering
- AI learning coach with simple step-by-step explanations

### ✅ My Daily Log
- Track daily accomplishments with quick-tap chips
- Persistent local storage — survives browser restarts

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML · CSS · Vanilla JavaScript |
| AI Engine | Groq API — Llama 3.3 70B · Llama 4 Scout (vision) |
| Voice | Web Speech API — built into Chrome · 100% free |
| Speech Input | Web Speech Recognition API |
| Hosting | Vercel (primary) · GitHub Pages (backup) |
| Storage | localStorage — no database needed |
| Cost | $0/month to run |

## 🏗️ Architecture

```
Single HTML file — no build step · no framework · no dependencies
        │
        ├── 11 tabs — all features in one file
        ├── Groq API — AI chat + script generation
        ├── Web Speech API — voice output (Becky & Claude)
        ├── Speech Recognition — mic input + live voice mode
        └── localStorage — API key · voice · memory · log
```
